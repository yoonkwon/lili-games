# 숲속 모험단 - 게임 스펙 문서

## 개요
- **장르**: 캐주얼 수집 모험 게임
- **컨셉**: 동물의 숲 스타일 수집 + 시간 연장 메카닉
- **타겟**: 4~8세 아동
- **플랫폼**: 웹 (HTML5 Canvas), PWA 지원
- **기술**: ES Modules, Canvas 2D, Web Audio API, shared/ 공용 모듈 활용

---

## 스토리

### 배경
리아(6살)와 리사(4살) 자매가 마법의 숲에 놀러 갔다가,
숲의 요정이 "숲이 잠들고 있어요... 반짝이 열매를 모아주면 숲이 다시 깨어날 거예요!"라고 부탁한다.
자매는 숲을 탐험하며 열매, 꽃, 조개, 별조각 등을 모으고,
길에서 만난 동료들과 함께 더 넓은 세계로 모험을 떠난다.

### 캐릭터

| 캐릭터 | 설명 | 특기 | 해금 조건 |
|--------|------|------|-----------|
| 🧒 리아 | 6살 언니, 용감하고 호기심 많음 | 빠른 이동 | 시작 캐릭터 |
| 👧 리사 | 4살 동생, 귀엽고 운이 좋음 | 희귀 아이템 확률 UP | 시작 캐릭터 |
| 🐕 보리 | 검은 차우차우, 듬직한 형 | 냄새로 숨은 아이템 탐지 | 라운드 2 클리어 |
| 🐶 좁쌀이 | 점박이 강아지, 활발함 | 넓은 범위 수집 | 라운드 3 클리어 |
| 🦕 익돌이 | 주황색 프테라노돈, 하늘의 친구 | 높은 곳 아이템 수집 | 라운드 5 클리어 |

## 핵심 메카닉

### 1. 타이머 & 라운드 시스템
| 항목 | 값 |
|------|-----|
| 라운드 시간 | 5분 (300초) |
| 연장 조건 | 라운드 목표 수집량 달성 |
| 연장 시간 | +5분 |
| 최대 라운드 | 무제한 (끝까지 도전) |
| 라운드 목표 | 라운드별 증가 (아래 표 참조) |

### 라운드별 목표 & 테마
| 라운드 | 장소 | 목표 | 수집 아이템 | 이벤트 |
|--------|------|------|-------------|--------|
| 1 | 🌸 꽃밭 초원 | 15개 | 꽃, 나비, 딸기 | 튜토리얼 |
| 2 | 🌲 깊은 숲 | 20개 | 버섯, 도토리, 반딧불이 | 보리 합류 |
| 3 | 🏖️ 반짝이 해변 | 25개 | 조개, 불가사리, 유리병 | 좁쌀이 합류 |
| 4 | 🏔️ 구름 산 | 30개 | 별조각, 무지개꽃, 크리스탈 | 바람 이벤트 |
| 5 | 🌌 별빛 하늘 | 35개 | 별똥별, 달조각, 오로라꽃 | 익돌이 합류 |
| 6+ | 🌈 무지개 세계 | 35+5n | 모든 아이템 랜덤 | 보너스 라운드 |

### 2. 수집 시스템
- **터치/클릭**으로 화면의 아이템을 수집
- 아이템은 맵 곳곳에 **랜덤 스폰**
- 일부 아이템은 **숨겨져 있음** (풀 뒤, 바위 뒤 → 터치로 발견)
- 수집 시 파티클 효과 + 사운드

### 아이템 등급
| 등급 | 가치 | 확률 | 시각 효과 |
|------|------|------|-----------|
| ⚪ 일반 | 1개 | 60% | 없음 |
| 🟡 반짝이 | 2개 | 25% | 반짝임 |
| 🟣 희귀 | 3개 | 12% | 빛 기둥 |
| 🌈 전설 | 5개 | 3% | 무지개 후광 |

### 3. 동료 시스템
- 동료는 **자동으로** 주변 아이템을 수집해줌
- 각 동료마다 고유 능력이 있음
- 동료가 많을수록 수집 효율 UP

| 동료 | 수집 범위 | 수집 속도 | 특수 능력 |
|------|-----------|-----------|-----------|
| 보리 | 80px | 3초/개 | 숨은 아이템 자동 발견 (30초마다) |
| 좁쌀이 | 150px | 2초/개 | 이동 속도 빠름, 넓은 범위 |
| 익돌이 | 200px | 4초/개 | 하늘 아이템 수집 (다른 동료 불가) |

### 4. 이동 시스템
- 화면 **터치한 방향**으로 캐릭터 이동
- 캐릭터 근처 아이템은 **자동 수집** (반경 50px)
- 동료들은 캐릭터를 **따라다님** (줄줄이)

### 5. 특수 이벤트
각 라운드에서 랜덤으로 발생:

| 이벤트 | 효과 | 확률 |
|--------|------|------|
| 🌟 별비 | 30초간 아이템 2배 스폰 | 15% |
| 🦋 나비 무리 | 나비 터치 시 보너스 +5 | 20% |
| 🌈 무지개 | 10초간 모든 아이템 등급 UP | 10% |
| 💨 바람 | 아이템이 흩어짐 (위치 변경) | 15% |
| 🎁 선물 상자 | 터치 시 보너스 +10 | 5% |

## UI 구성

### 게임 화면
```
┌─────────────────────────────┐
│ ⏰ 4:32  │ 🧺 12/20  │ R2  │  ← 타이머, 수집현황, 라운드
│                             │
│     (수집 아이템들이         │
│      맵 곳곳에 배치)        │
│                             │
│        🧒👧                 │  ← 리아 + 리사 (+ 동료들)
│        🐕🐶                 │
│                             │
│  [아이템 인벤토리 미니뷰]    │  ← 하단

```

### 씬 구성
| 씬 | 설명 |
|----|------|
| 타이틀 | 게임 시작, 도감 보기 |
| 인트로 | 스토리 컷씬 (첫 플레이만) |
| 게임 | 메인 수집 플레이 |
| 라운드 클리어 | 축하 + 동료 해금 연출 |
| 게임오버 | 시간 초과, 기록 표시 |
| 도감 | 수집한 아이템 목록 |

## 점수 시스템

```
점수 = 일반×10 + 반짝이×25 + 희귀×50 + 전설×100 + 라운드보너스×200
```

### 별 등급 (라운드 종료 시)
| 등급 | 조건 |
|------|------|
| ⭐⭐⭐ | 목표의 150% 이상 수집 |
| ⭐⭐ | 목표의 120% 이상 수집 |
| ⭐ | 목표 달성 |

---

## 도감 시스템
- 수집한 아이템은 **영구 도감**에 기록 (localStorage)
- 전체 아이템 종류: 라운드별 3종 × 4등급 = 약 60종
- 도감 완성도에 따른 보상:
  - 25% → 리사 이동속도 UP
  - 50% → 시작 시 보너스 시간 +30초
  - 75% → 희귀 아이템 확률 2배
  - 100% → 전설 테두리 해금

## 개발 우선순위

### Phase 1 - 코어 (MVP)
1. 타이틀 씬 + 캐릭터 선택
2. 게임 씬 기본 (이동 + 수집 + 타이머)
3. 라운드 1 꽃밭 초원
4. 라운드 클리어 / 게임오버

### Phase 2 - 확장
5. 라운드 2~3 + 동료 시스템 (보리, 좁쌀이)
6. 아이템 등급 시스템
7. 특수 이벤트

### Phase 3 - 완성
8. 라운드 4~5 + 익돌이
9. 도감 시스템
10. 사운드 + 이펙트 풀리시
