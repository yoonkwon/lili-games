# 리리탐험대 - 게임 스펙 문서

## 개요
- **장르**: 교육형 도감 탐험 + 스무고개 퀴즈
- **컨셉**: 자유 탐험으로 한글·영어·숫자·공룡·동물 도감을 채우는 동물의 숲 스타일 캐주얼 학습 게임
- **타겟**: 4~8세 아동
- **플랫폼**: 웹 (HTML5 Canvas), PWA 지원
- **기술**: ES Modules, Canvas 2D, Web Audio API, devicePixelRatio 대응, shared/ 공용 모듈

---

## 스토리

리아(6살)와 리사(4살) 자매가 신비한 숲에 들어가서 한글·영어·숫자·공룡·동물의 비밀을 찾는다.
숲 곳곳에 숨어있는 글자와 동물을 발견하고, 동료 친구들과 함께 단어를 완성하거나 스무고개 단서를 풀어 낸다.

### 캐릭터

| 캐릭터 | 설명 | 역할 |
|--------|------|------|
| 🧒 리아 | 6살 언니, 메인 플레이어 | 직접 조작 (탭한 곳으로 이동) |
| 👧 리사 | 4살 동생, 버디 | 리아를 따라다님 |
| 🐕 보리 | 검은 차우차우 | 숨은 아이템 탐지 |
| 🦊 좁쌀이 | 빠른 강아지 | 대시로 빠르게 수집 |
| 🐱 고순이 | 행운의 고양이 | 럭키 아우라로 특별 아이템 발견 |
| 🦅 익돌이 | 하늘의 친구 | 하늘에서 내려와 수집 |
| 🐰 아찌 쌍둥이 (흰/하늘) | 함께 다니는 토끼 | 짝꿍 탐지 |

라운드가 진행되며 컴패니언이 셔플 순서로 합류한다.

---

## 게임 모드

### 1. 탐험 모드 (Explore)
시간 제한 없는 자유 탐험. 맵에 흩뿌려진 학습 아이템을 모두 발견하면 라운드 클리어.

### 2. 스무고개 퀴즈 (Quiz)
🔍 수사 포인트를 모으면 가린 단서가 풀린다. 단서가 충분히 모이면 4지선다 정답 맞히기.

## 스테이지 구성

### 탐험 스테이지 (`STAGES`)
| 스테이지 | 테마 | 아이템 | 특징 |
|----------|------|--------|------|
| 🇰🇷 한글 나라 | 푸른 숲 | 자음 14개 + 모음 6개 | 단어 만들기 미션 (음절 조립) |
| 🔤 영어 나라 | 따뜻한 들판 | 알파벳 26개 | 단어 만들기 미션 (CAT, DOG…) |
| 🔢 숫자 나라 | 마법 들판 | 0~10 + 100 | 숫자 도감 |
| 🦕 공룡 나라 | 선사시대 | 공룡 12종 (PNG 스프라이트) | 멸종 동물 도감 |
| 🐾 동물 나라 | 정글 | 동물 14종 | 신기한 동물 도감 |

### 퀴즈 스테이지 (`QUIZ_STAGES`)
| 스테이지 | 형식 |
|----------|------|
| 🔍 동물 스무고개 | 단서 5개 → 4지선다 / 라운드별 셔플 |

## 핵심 메카닉

### 1. 발견(Discovery) 시스템
- 맵에 아이템이 풀, 바위, 나무, 꽃, ✨ 반짝임 등으로 위장돼 있다
- 플레이어가 가까이 가면 (HINT_RADIUS=150) 살짝 보이고, 더 가까이(DISCOVER_RADIUS=80) 가면 ❓ 표시 + 탭 가능
- 탭하면 발견 → 도감 등록 + ✨파티클 + 컴패니언 반응 대사
- 발견 팝업: 큰 이모지/스프라이트 + 이름 + 재미있는 사실(funFact) + 확인 버튼

### 2. 단어 만들기 미션 (`WORD_MISSIONS`)
- **한글**: 자모 4~5개를 모두 발견하면 미션 등장. 슬롯에 음절 블록(예: `토`, `끼`) 두 개를 순서대로 놓으면 완성
- **영어**: 알파벳 글자 발견 후 단어 슬롯에 글자 조립
- 풀에는 정답 + 디스트랙터 2~3개 (한글은 `HANGUL_SYLLABLES` 풀, 영어는 `ENGLISH_LETTERS` 풀에서 추출)
- 완성 시 🎉 단어 + 동물 이모지 + 재미있는 사실 표시

### 3. 컴패니언 퀘스트 (`COMPANION_QUESTS`)
스테이지마다 정해진 목표 (특정 글자 모으기, n개 모으기 등)를 완료하면 컴패니언 대사 + 별 보상.

### 4. 컴패니언 능력
| 컴패니언 | 능력 | 주기 |
|----------|------|------|
| 보리 | 숨은 아이템 자동 탐지 | 8초 |
| 좁쌀이 | 빠른 대시로 수집 | 10초 |
| 고순이 | 행운 아우라 | 15초 |
| 익돌이 | 하늘 스왑 | 12초 |
| 아찌 쌍둥이 | 짝꿍 탐지 | 9초 |

각 컴패니언은 라운드별로 셔플 순서대로 등장.

### 5. 스무고개 퀴즈
- 🔍 수사 포인트 두 종류 수집:
  - 첫 번째: 새 단서 문장 등장 (핵심 단어 가린 채)
  - 두 번째: 가린 단어 공개
- 단서 3개 이상 풀리면 "🤔 맞춰보기!" 버튼 활성화
- 4지선다, 컴패니언이 방향/단서 힌트 제공

## UI / 입력

- **이동**: 화면 탭 → 그 위치로 리아가 이동
- **수집**: 아이템 위 탭 → 발견
- **팝업/오버레이**: 발견 팝업, 단어 만들기, 퀴즈, 정답/오답 화면 등은 입력을 인터셉트
- **safe-area-inset 대응**: 캔버스 좌표 변환은 `getBoundingClientRect()` 기반 (iPhone 노치/Dynamic Island)
- **devicePixelRatio**: 캔버스 버퍼를 `cssW × DPR`로 설정해 레티나 디스플레이에서 선명하게 렌더

---

## 도감 / 저장

- **localStorage**: 진행 중 자동 세이브 (5초 주기), 발견 아이템 영구 보관
- **도감 (Encyclopedia)**: 스테이지별 발견 글자/동물, 완성한 단어를 영구 기록
- **이어하기**: 타이틀에서 마지막 진행 상태 복원

---

## 기술 노트

- **SpriteCache**: 리아/리사 PNG는 3× 해상도(192px tall)로 캐시하고 `_displayW/_displayH`로 다운스케일 → 디테일 보존 + 레티나 선명
- **공룡/해마**: 사전 렌더 PNG (`assets/sprites/dino-*.png`, `animal-seahorse.png`)
- **다른 동물·자모·알파벳**: 시스템 이모지 폰트로 직접 렌더 (`drawSpriteOrEmoji`)
- **터레인**: `TERRAIN_PRESETS`로 스테이지별 프로시저럴 배경 생성 (forest, jungle, magical, prehistoric)
