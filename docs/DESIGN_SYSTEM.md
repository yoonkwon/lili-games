# 꼬꼬닭 알 낳기 - 디자인 시스템

> 5~6세 아동 대상. 읽기 불가 전제. 모든 UI는 아이콘+사운드로 전달.

---

## 1. 디자인 원칙

1. **글자 없이 이해** - 텍스트는 장식. 아이콘/소리/캐릭터 반응이 정보 전달 수단
2. **실패 없는 경험** - 벌점/감점 없음. 적에게 알을 빼앗겨도 "다시 모으자!" 분위기
3. **즉각 반응** - 터치 후 200ms 이내 피드백 (소리 + 시각)
4. **크고 선명하게** - 터치 영역 최소 72px, 주요 버튼 100px 이상
5. **단순한 화면** - 한 화면에 인터랙티브 요소 최대 3~4개

| 역할 | 색상 | HEX | 용도 |
|------|------|-----|------|

| 부리/발 | 주황 | `#FF8C00` | 부리, 발, CTA 버튼 |
| 금알 | 골드 | `#FFD700` | 금알, 보상 하이라이트 |
| 하늘 | 하늘색 | `#87CEEB` | 배경 하늘 |
| 잔디 | 초록 | `#7EC850` | 바닥 초원 |
| 울타리 | 나무색 | `#DEB887` | 울타리 |
| 바구니 | 짙은 나무 | `#B8860B` | 바구니 |

### 배경 규칙
- 하늘: 부드러운 그라데이션 (`#87CEEB` → `#E0F0FF`)
- 잔디: 부드러운 그라데이션 (`#7EC850` → `#4A8B2C`)
- 채도 낮은 파스텔 톤으로 눈 보호

### 인터랙티브 요소 규칙
- **터치 가능한 것**: 따뜻한 색 (주황/노랑/빨강) + 그림자 + 약간의 바운스
- **배경/장식**: 차가운 색 (파랑/초록) + 채도 낮음
- 한 화면에 **주 색상 3~4개** 이하

│                                              │
│                                              │ ← 중앙: 게임 영역

## 5. 터치 영역

| 요소 | 시각 크기 | 터치 영역 | 비고 |
|------|----------|----------|------|
| 화면 전체 (알 낳기) | 전체 | 전체 | 어디든 터치 = 게이지 +1 |
| 적 캐릭터 | 90×55 | 120×90 | 시각보다 30% 넓게 |
| 시작하기 버튼 | 220×65 | 240×85 | 패딩 10px 추가 |
| 다시하기 버튼 | 200×55 | 220×75 | 패딩 10px 추가 |

### 터치 피드백
- **닭 터치** (게이지 펌프): 즉시 squish 애니메이션 + 꼬꼬 소리 (< 50ms)
- **적 터치**: 즉시 도망 애니메이션 + 환호 소리 + 파티클
- **버튼 터치**: scale(0.9) → scale(1.05) → scale(1) (150ms)

- **볼륨 위계**: 꼬꼬(터치) > 알 낳기 > 배경

## 13. 반응형 처리

| 화면 | 조정 |
|------|------|
| 모바일 세로 (< 480px 너비) | 게이지 너비 90%, 닭/바구니 세로 배치 |
| 태블릿 가로 (768~1024px) | 기본 레이아웃 |
| 데스크탑 (> 1024px) | 최대 너비 1200px 제한, 중앙 정렬 |

### 공통
- `user-scalable=no` (핀치줌 방지)
- `touch-action: none` (기본 제스처 차단)
- 화면 리사이즈 시 캔버스 재계산
- 모든 좌표 **비율(%) 기반** + 최소값 보장
