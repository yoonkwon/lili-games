# 꼬꼬닭 알 낳기 - 게임 스펙 문서

## 개요
- **장르**: 캐주얼 터치/클릭 게임
- **타겟 연령**: 5~6세 아동
- **플랫폼**: 웹 (HTML5 Canvas)
- **기술 스택**: 단일 HTML 파일 (Vanilla JS, Canvas 2D, Web Audio API)
- **서버**: `npx http-server` (0.0.0.0:8080)

---

## 게임 목표
알 **100개**를 바구니에 모아 **병아리 100마리를 부화**시켜 엄마닭과 행복한 엔딩을 본다.

---

## 핵심 메카닉

### 1. 게이지 시스템
| 항목 | 값 |
|------|-----|
| 게이지 최대치 | 8칸 |
| 채우는 방법 | 화면 터치/클릭 1회 = 1칸 |
| 자동 감소 | 없음 (터치할 때만 증가) |
| 게이지 만땅 시 | 알 1개 생성 → 바구니로 자동 비행 |
| 금알 확률 | 12% |
| 표시 | 화면 상단 게이지 바 + 숫자 (예: `3 / 8`) |

### 2. 알 비행
- 게이지가 차면 닭에서 알이 생성됨
- **베지어 곡선(포물선)**으로 바구니까지 자동 비행
- 도착 시 바구니 카운트 +1 & 파티클 이펙트

### 3. 적 시스템 (도둑)
| 종류 | 색상 | 이모지 |
|------|------|--------|
| 여우 | 주황 (#FF8C42) | 🦊 |
| 족제비 | 갈색 (#8B6914) | 🐾 |
| 너구리 | 회색 (#808080) | 🦝 |

- **출현 조건**: 바구니 알 5개 이상
- **출현 간격**: `max(6초, 15초 - 바구니알수 × 0.08초)` (알이 많을수록 자주 등장)
- **행동 패턴**: 좌 또는 우에서 등장 → 바구니로 접근 → 0.8초 후 알 훔침 → 도주
- **도둑질 양**: `min(3, max(1, 바구니알수 × 20%))` 개
- **방어**: 적 캐릭터를 터치하면 쫓아냄 (바구니 도착 전에 터치해야 함)
- **경고**: 바구니에서 250px 이내 접근 시 메시지 표시 ("🦊 여우다! 터치해서 쫓아내!")
- **적 머리 위 ❗ 표시**로 접근 중임을 알림

## 씬 (Scene) 구성

### 1. 타이틀 (`title`)
- 게임 제목: "꼬꼬닭 알 낳기"
- 부제: "🥚 알 100개를 모아 병아리를 부화시켜요! 🐣"
- 닭 캐릭터 애니메이션 (바운스 + 날개 퍼덕임)
- "시작하기! 🎮" 버튼

### 2. 게임 플레이 (`game`)
- 배경: 하늘 + 구름 + 태양 + 초원 + 울타리 + 꽃
- 닭 캐릭터 (화면 좌측 35%)
- 바구니 (화면 우측)
- 게이지 바 (화면 상단 중앙)
- 정보 바: 알 수 (`🥚 n / 100`), 진행 바, 병아리 수

### 3. 엔딩 (`ending`)
- 엄마닭 중앙에 배치 (날개 벌림)
- 병아리 100마리가 원형으로 둘러쌈 (초당 15마리씩 등장 애니메이션)
- 하트 ❤️ 떠다니는 이펙트
- "🎉 축하해! 🎉" + "엄마닭과 병아리 N마리가 행복하게 살았답니다!"
- 5초 후 "다시 하기! 🔄" 버튼 표시

## 캐릭터 디자인

### 엄마닭 (Chicken)
- **몸통**: 타원형, 그라데이션 (#FFE4B5 → #FFCC66 → #E8A317)
- **머리**: 타원형, 볏 3개 (빨간색 #FF4444)
- **눈**: 흰자 + 검은 동공 + 하이라이트, 행복할 때 반짝이는 큰 눈
- **부리**: 다이아몬드형 주황색 (#FF8C00), 아래쪽 어두운 톤
- **볼터치**: 양쪽 연분홍 (rgba(255,150,150,0.35))
- **날개**: 양쪽 타원, 펄럭임 애니메이션
- **꼬리깃**: 3개, 좌우 흔들림
- **발**: 주황색, 각 3개 발가락
- **턱 고기(Wattle)**: 빨간색 타원
- **눈 깜빡임**: 3~5초 간격, 150ms
- **모자 슬롯**: 왕관/리본/꽃 장착 가능

### 병아리 (Chick)
- **몸통/머리**: 노란 계열 그라데이션
- **머리 위 깃**: 작은 타원
- **눈**: 작은 검은 점 + 하이라이트
- **부리**: 작은 삼각형
- **볼터치**: 연분홍
- **날개**: 작은 타원, 미세 펄럭임
- **행동**: 랜덤 걸어다님 (2~5초 간격), 삐약삐약 (4~9초 간격, 말풍선)
- **색상 바리에이션**: 10종 (#FFE44D, #FFD700, #FFC125, #FFAA00, #FFB347, #FF8C69, #DDA0DD, #98FB98, #87CEEB, #FFB6C1)

### 적 캐릭터 공통
- 몸통 + 배 (밝은 톤) + 머리 + 귀 (내부 색) + 주둥이 + 꼬리 (끝 색 다름)
- 겁먹으면 눈이 × 로 변경, 반대 방향으로 도주
- 너구리: 눈 주위 검은 마스크 + 꼬리 줄무늬 추가

## 애니메이션 & 이펙트

### 닭 애니메이션
| 상태 | 동작 |
|------|------|
| 대기 (phase 0) | 상하 bobble (sin파 × 3px) |
| 알 낳기 (phase 2) | squish 0.35 → 0으로 감소 (0.35초), 날개 펄럭 |

### 파티클
- 알 낳을 때: 닭 아래에서 12개 생성
- 바구니 도착 시: 8개 생성
- 적 쫓아낼 때: 금색 10개 생성
- 적 알 훔칠 때: 빨간색 8개 생성
- 물리: 중력 300, 수명 감소 dt×2

### 플로팅 텍스트
- "+1" 또는 "+1 ⭐" (금알): 위로 떠오르며 페이드아웃

### 메시지 시스템
- 화면 중앙 상단에 2.5초간 표시
- 반투명 검은 배경 + 흰 글씨 + 둥근 모서리

## 사운드 (Web Audio API 합성음)

| 효과음 | 설명 |
|--------|------|
| playCluck() | 꼬꼬꼬 - 터치마다 재생 (600→500→400Hz square) |
| playEggPop() | 알 낳기 (800→1000→1200Hz sine) |
| playGoldenEgg() | 금알 (600~1400Hz 5음 상승) |
| playCheer() | 환호 (도-미-솔-높은도) |
| playPeep() | 병아리 삐약 (1800→2000Hz sine) |
| playScared() | 적 도망 (200→150Hz sawtooth) |
| playSteal() | 알 도둑질 (300→200Hz square) |
| playEnding() | 엔딩 멜로디 (도레미파솔라시도, 8음) |

| 요소 | 수량 | 설명 |
|------|------|------|
