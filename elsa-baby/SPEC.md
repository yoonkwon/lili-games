# 엘사 아기 키우기 - 게임 스펙 문서

## 1. 게임 개요

- **이름**: 엘사 아기 키우기
- **컨셉**: 컨베이어 벨트 위를 지나가는 음식 중 아기가 원하는 음식을 골라 먹여 아기를 성장시키는 터치 기반 캐주얼 게임
- **타겟 유저**: 어린이 (넉넉한 터치 판정, 이모지 기반 UI, 한글 안내)
- **테마**: 디즈니 겨울왕국(Frozen) 스타일 (어두운 밤하늘 배경, 눈송이, 얼음 마법)

---

## 2. 게임 메카닉

### 2.1 컨베이어 벨트 시스템

화면 하단에 컨베이어 벨트가 좌에서 우로 흐르며, 그 위에 음식이 접시에 담겨 이동한다. 플레이어는 음식을 터치하여 아기에게 먹인다.

**벨트 구성**
- 벨트 위치: 화면 세로 85% (`_getBeltY`)
- 벨트 높이: 80px (메탈릭 레일 상하단 포함)
- 아이템 크기: `min(60, 화면너비 x 0.14)` px
- 아이템 간격: 75px (`itemSpacing`)
- 이동 속도: 55px/초 (`conveyorSpeed`)
- 이동 방향: 좌 → 우

**스폰 규칙**
- 초기: 화면 너비를 itemSpacing으로 나눈 수 + 2개를 균등 배치
- 이후: `itemSpacing / conveyorSpeed` 간격(약 1.36초)마다 좌측(-60px)에서 새 아이템 스폰
- 화면 우측 밖(w + 80px)으로 나가면 제거

**아이템 접시 렌더링**
- 원형 접시 배경 (#2a3a5e → #354a70)
- 접시 테두리 (반투명 흰색)
- 음식 이모지 + 이름 라벨

**수집 애니메이션**
- 터치 시 위로 날아가며(`-60px`) 페이드 아웃 + 확대(1.5배)

**음식 목록 (12종)**

| 딸기 | 🍓 | #FF4444 |
| 초콜릿 | 🍫 | #8B4513 |
| 아이스크림 | 🍦 | #FFE4B5 |

| 수박 | 🍉 | #2E8B57 |

### 2.2 음식 선택 메카닉

**원하는 음식 표시**
- 아기가 원하는 음식은 말풍선으로 표시 (화면 중앙 우측 상단, `w/2 + 50`, `h * 0.28`)
- 말풍선 내부에 음식 이모지가 맥동(pulse) 애니메이션으로 표시
- 변경 주기: 8초 (`wantChangeInterval`)

**타이머 긴급도 표시**
- 40% 경과 시: "기다려..." (주황색)
- 70% 경과 시: "빨리!" (빨간색)
- 타이머 만료 시: 싫음 +8, 감정 sad (0.8초), 새 음식 선택

**컨베이어 음식 확률**
- 40% 확률로 아기가 원하는 음식 스폰
- 60% 확률로 12종 중 랜덤 음식 스폰

**정답 음식 터치 시**
- 성장치 + (8 + 콤보수 x 2) (`growthPerCorrect` + `combo` x `comboBonus`)
- 싫음 게이지 -10
- 아기 감정: happy (1.5초)
- 금색 파티클(12개) + 별 이펙트(5개) + "+N 💕" 플로팅 텍스트
- 새로운 원하는 음식 선택

**오답 음식 터치 시**
- 성장치 -2 (`growthPerWrong`)
- 싫음 게이지 +20 (`dislikePerWrong`)
- 콤보 초기화
- 아기 감정: sad (1.2초) + 흔들림 효과 (0.5초)
- "싫어요! 😣" 플로팅 텍스트

### 2.3 성장 시스템

- **최대 성장치**: 100 (`maxGrowth`)
- 성장치가 100에 도달하면 아기 탄생 (Born 씬으로 전환)

**성장 단계 (6단계)**

| 단계명 | 크기 배율 | 진행률 기준 | 설명 |
|--------|----------|-----------|------|
| 콩알 | 0.3 | 0% | 아주 작은 아기에요 |
| 새우 | 0.45 | 15% | 조금 커졌어요! |
| 레몬 | 0.6 | 30% | 점점 자라고 있어요! |
| 사과 | 0.75 | 50% | 많이 컸어요! |
| 멜론 | 0.9 | 70% | 거의 다 자랐어요! |
| 아기 | 1.0 | 90% | 세상에 나올 준비! |

**아기 크기 공식**: `20 + (성장비율 x 35)` px

**배(belly) 원 크기**: `60 + (성장비율 x 40)` px 반지름

### 2.4 싫음 게이지 & 얼음 프리즈

- **최대 싫음**: 100 (`maxDislike`)
- **자연 감소**: 초당 3 (`dislikeDecay`)
- **싫음 바 위치**: 성장 바 아래 (화면 너비의 40%)
- **싫음 70% 이상 시**: 빨간색 깜빡임 경고 효과

**얼음 마법 (싫음 100 도달 시)**
- 지속 시간: 5초 (`freezeDuration`)
- 효과: 입력 차단 (터치 무시), 컨베이어 벨트 정지
- 아기 감정: angry (프리즈 지속 동안)
- 메시지: "❄️ 아기가 얼음 마법을 썼어요! ❄️"
- 시각 효과:
  - 연청색 오버레이 (`rgba(173, 216, 250, 0.6)`)
  - 가장자리 라디얼 그라디언트
  - 얼음 균열(ice crack) 8개 (랜덤 경로)
  - 회전하는 얼음 이모지 12개 (❄️, 🧊, ❄️, 💎)
  - 중앙 카운트다운 표시 ("❄️ N초 ❄️")
- 종료 시: 싫음 0으로 초기화, 감정 neutral 복구, 새 음식 선택

### 2.5 콤보 시스템

- 정답 연속 터치 시 콤보 카운트 증가
- 콤보 보너스: 콤보수 x 2 추가 성장치
- 3콤보 이상 시 메시지 표시: "N 콤보! 아기가 아주 좋아해요! ✨"
- 2콤보 이상 시 화면에 "⚡ N 콤보!" 표시 (화면 세로 68% 위치, 금색)
- 오답 터치 시 콤보 초기화

## 3. 캐릭터

### 3.1 엄마 엘사

- **에셋**: `assets/elsa-mom.svg` (SVG 이미지)
- **렌더링 크기**: 100 x 150px (scale 적용)
- **위치**: 타이틀 씬 - 화면 상단 20% (scale 1.3), 게임 씬 - 화면 상단 14% (scale 1.0)
- **애니메이션**: 상하 바운스 (sin파, 진폭 3~5px)
- **라벨**: "엄마 엘사" (게임 씬)

### 3.2 아기 엘사 (4종 감정 표현)

| 감정 | 에셋 파일 | 트리거 |
|------|----------|--------|
| neutral | `assets/baby-elsa.svg` | 기본 상태 |
| happy | `assets/baby-elsa-happy.svg` | 정답 음식 터치 (1.5초) |
| sad | `assets/baby-elsa-sad.svg` | 오답 음식 터치 (1.2초), 타이머 만료 (0.8초) |
| angry | `assets/baby-elsa-angry.svg` | 얼음 프리즈 발동 (프리즈 지속 동안) |

- **위치**: 배 원(belly) 중앙 (화면 세로 40%)
- **효과**:
  - 성장비율 50% 이상 시 헤어 오버레이 글로우 (투명도 점진 증가)
  - 성장비율 60% 이상 시 ❄️ 눈송이 스파클 (반짝임)
- **흔들림**: 오답 시 좌우 흔들림 (sin 30Hz, 0.5초 감쇠)

## 4. 씬 구성

### 4.1 TitleScene (타이틀 화면)

- **배경**: 마법의 밤하늘 그라디언트 (#0d1b3e → #1a3a5c → #2a5298 → #4fc3f7)
- **장식**: 떠다니는 눈송이 30개 (❄️, 다양한 크기/속도/흔들림)
- **구성 요소**:
  - 엄마 엘사 (scale 1.3, 바운스 애니메이션)
  - 주변 회전 스파클 5개 (✨, 반지름 55px)
  - 제목: "❄️ 엘사 아기 키우기 ❄️" (38px Bold, 하늘색 글로우)
  - 부제: "엄마 엘사가 아기에게 음식을 줘요!" / "좋아하는 음식을 맞춰보세요~ 💕"
  - 아기 미리보기 (하늘색 글로우 원 안, neutral, 성장비율 0.3)
  - 시작 버튼: "❄️ 시작하기 ❄️" (하늘색→파란색 그라디언트, 맥동 효과, 둥근 모서리 22px)
  - 하단 힌트: "아기가 원하는 음식을 골라주세요!"

### 4.2 GameScene (게임 플레이)

- **배경**: 어두운 밤하늘 그라디언트 (#0d1b3e → #1a3a5c → #2a5298) + 눈송이 15개
- **UI 구성**:
  - 상단: 성장 바 (핑크 그라디언트 #FF69B4 → #FF1493, 퍼센트 표시)
  - 상단 아래: 싫음 바 (하늘색 그라디언트 #4fc3f7 → #29B6F6, 70% 이상 시 빨간 깜빡임)
  - 엄마 엘사 (화면 상단 14%, "엄마 엘사" 라벨)
  - 원하는 음식 말풍선 (엄마 우측)
  - 배 원 + 아기 (화면 세로 40%, 반짝이 장식 6개)
  - 성장 단계 텍스트
  - 콤보 표시 (2콤보 이상)
  - 컨베이어 벨트 안내: "🍽️ 아기가 원하는 음식을 터치!"
  - 컨베이어 벨트 + 음식 아이템 (화면 하단 85%)
- **파티클 시스템**: 정답/오답 시 이펙트
- **메시지 시스템**: 이벤트 알림 (초기 안내 3초 표시)

### 4.3 BornScene (아기 탄생)

- **배경**: 밤하늘~핑크 그라디언트 (#1a1a3e → #2a3a6e → #4a2a5e → #FF69B4)
- **축하 파티클**: 하트/별/눈송이 25개 (💕❤️💖✨🌟❄️, 하단에서 상단으로 부유)
- **페이즈 연출** (시간 기반):
  - 0.5초: 중앙 빛 효과 (라디얼 그라디언트)
  - 1.5초: 엄마 엘사(scale 1.2) + 아기(happy, 성장비율 1.0) 등장, 💕 하트
  - 2.5초: "🎉 아기가 태어났어요! 🎉" + "엄마 품에서 행복해하고 있어요 💕"
  - 3.5초: 통계 표시 (N번째 아기 뱃지, 총 먹인 횟수, 정확도, "사랑으로 키웠어요!")
  - 4.0초: "다시 하기 💕" 버튼 (핑크 그라디언트 #FF69B4 → #E91E63, 맥동)

- **데이터 저장**: `localStorage`에 누적 출생 횟수 저장 (키: `elsa-baby-born-count`)
- **통계 항목**: 총 먹인 횟수 (`totalFed`), 정답 횟수 (`correctFed`), 정확도 (`accuracy`)

## 6. 기술 스택

| 항목 | 기술 |
|------|------|
| 렌더링 | HTML5 Canvas 2D (`<canvas>`) |
| 모듈 시스템 | ES Modules (`type="module"`) |
| 캐릭터 에셋 | SVG 이미지 (5종: 엄마 1 + 아기 4) |
| 게임 루프 | `requestAnimationFrame` (dt 기반, 최대 50ms 캡) |
| 입력 처리 | `shared/Input.js` (터치/클릭 추상화) |
| 파티클 | `shared/ParticleSystem.js` |
| 메시지 UI | `shared/ui/Message.js` |
| 화면 전환 | Fade in/out (alpha 기반, 속도 2.5, 최대 불투명도 0.7) |
| 데이터 저장 | `localStorage` (출생 횟수) |
| 반응형 | 전체 화면, `resize` 이벤트 대응, safe-area-inset 지원 |
| 모바일 지원 | PWA manifest, apple-mobile-web-app-capable, touch-action: none, user-select: none |

## 7. 파일 구조

```
elsa-baby/
├── index.html                      # 진입점 HTML (캔버스, 모듈 로드)
├── manifest.json                   # PWA 매니페스트
├── assets/
│   ├── elsa-mom.svg               # 엄마 엘사 SVG
│   ├── baby-elsa.svg              # 아기 (neutral)
│   ├── baby-elsa-happy.svg        # 아기 (happy)
│   ├── baby-elsa-sad.svg          # 아기 (sad)
│   └── baby-elsa-angry.svg        # 아기 (angry)
└── js/
    ├── main.js                     # 메인 엔트리, 게임 루프, 씬 관리, 전환
    ├── config.js                   # 모든 게임 설정값 (음식, 성장 단계, 밸런스)
    ├── draw-elsa.js                # SVG 에셋 로딩 및 캐릭터 렌더링 함수
    └── scene/

        ├── GameScene.js            # 메인 게임 플레이 (컨베이어 벨트, 성장, 얼음 프리즈)
        └── BornScene.js            # 아기 탄생 결과 화면 (통계, 누적 카운트)

shared/                             # 프로젝트 공통 모듈 (상위 디렉토리)
├── Input.js                        # 터치/클릭 입력 처리
├── ParticleSystem.js               # 파티클 이펙트
└── ui/
    └── Message.js                  # 화면 메시지 표시
```
