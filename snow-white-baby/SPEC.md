# 백설공주 아기 키우기 - 게임 스펙 문서

## 1. 게임 개요

- **이름**: 백설공주 아기 키우기
- **컨셉**: 숲속 동물들이 음식을 배달해주면, 아기가 원하는 음식을 골라서 먹여 아기를 성장시키는 터치 기반 캐주얼 게임
- **타겟 유저**: 어린이 (넉넉한 터치 판정, 이모지 기반 UI, 한글 안내)
- **테마**: 디즈니풍 백설공주 + 숲속 동화 분위기 (어두운 숲 배경, 반딧불, 낙엽)

---

## 2. 게임 메카닉

### 2.1 동물 택배 시스템

숲속 동물들이 화면 좌우에서 등장하여 음식을 들고 온다. 플레이어는 동물을 터치하여 아기에게 음식을 먹인다.

**동물 종류 (4종)**

| 동물 | 이모지 | 이동속도 | 크기 |
|------|--------|---------|------|
| 새 | 🐦 | 120 | 40 |
| 토끼 | 🐰 | 80 | 50 |
| 사슴 | 🦌 | 50 | 65 |
| 다람쥐 | 🐿️ | 110 | 42 |

**스폰 규칙**
- 화면 좌측(-50px) 또는 우측(w+50px)에서 출현, y좌표는 화면 하단 65~85% 영역
- 스폰 간격: 2초 (`animalSpawnInterval`)
- 최대 동시 출현: 5마리 (`maxAnimals`)
- 최소 유지: 3마리 (`minAnimals`) - 활성 동물이 부족하면 즉시 추가 스폰
- 목표 지점에 도착 후 8초간 머무르다 자동 퇴장

**음식 선택**
- 50% 확률로 아기가 원하는 음식을 가져옴 (코드상 `_pickAnimalFood` 기준, config의 `wantedFoodChance: 0.4`와 별도)
- 나머지는 12종 음식 중 랜덤

**음식 목록 (12종)**

| 음식 | 이모지 |
|------|--------|
| 사과 | 🍎 |
| 딸기 | 🍓 |
| 포도 | 🍇 |
| 체리 | 🍒 |
| 빵 | 🍞 |
| 치즈 | 🧀 |
| 파이 | 🥧 |
| 꿀 | 🍯 |
| 우유 | 🥛 |
| 버섯 | 🍄 |
| 블루베리 | 🫐 |
| 당근 | 🥕 |

### 2.2 마녀 시스템 (독사과 유혹)

- **등장 주기**: 15초마다 (`crowInterval`)
- **외형**: 마녀 이모지(🧙‍♀️), 보라색 배경 원, 보라색 경고 테두리 맥동 효과
- **유혹 메카닉**: 마녀는 아기가 현재 원하는 음식의 이모지를 위장하여 들고 옴 (겉보기에는 원하는 음식처럼 보임)
- **체류 시간**: 5초 후 자동 퇴장 (`stayDuration`)
- **이동 속도**: 70 / 크기: 55

**독사과 터치 시 페널티**
- 성장치 -15 (`poisonPenalty`)
- 싫음 게이지 +40 (`poisonDislike`)
- 콤보 초기화
- 아기 감정: angry (2초간)
- 보라색 파티클 + "독사과! ☠️" 메시지

### 2.3 성장 시스템

- **최대 성장치**: 100 (`maxGrowth`)
- 성장치가 100에 도달하면 아기 탄생 (Born 씬으로 전환)

**정답 음식 터치 시**
- 성장치 + (8 + 콤보수 x 2) (`growthPerCorrect` + `combo` x `comboBonus`)
- 싫음 게이지 -10
- 아기 감정: happy (1.5초)
- 금색 파티클 + 별 이펙트 + "+N 🌸" 텍스트
- 새로운 원하는 음식 선택

**오답 음식 터치 시**
- 성장치 -2 (`growthPerWrong`)
- 싫음 게이지 +20 (`dislikePerWrong`)
- 콤보 초기화
- 아기 감정: sad (1.2초) + 흔들림 효과 (0.5초)

**성장 단계 (6단계)**

| 단계명 | 크기 배율 | 진행률 기준 | 설명 |
|--------|----------|-----------|------|
| 콩알 | 0.3 | 0% | 아주 작은 아기에요 |
| 새우 | 0.45 | 15% | 조금 커졌어요! |
| 레몬 | 0.6 | 30% | 점점 자라고 있어요! |
| 사과 | 0.75 | 50% | 많이 컸어요! |
| 멜론 | 0.9 | 70% | 거의 다 자랐어요! |
| 아기 | 1.0 | 90% | 세상에 나올 준비! |

### 2.4 싫음 게이지 & 마법 거울 저주

- **최대 싫음**: 100 (`maxDislike`)
- **자연 감소**: 초당 3 (`dislikeDecay`)
- **원하는 음식 타이머 만료 시**: 싫음 +8, 새 음식 선택
- **싫음 100 도달 시**: 마법 거울 저주 발동

**마법 거울 저주**
- 지속 시간: 5초 (`curseDuration`)
- 효과: 입력 차단 (터치 무시), 동물 스폰/이동 정지
- 시각 효과: 어두운 보라색 오버레이, 타원형 거울 프레임, 저주 이모지(🪞💀👑🍎) 회전, 카운트다운 표시
- 종료 시: 싫음 0으로 초기화, 감정 neutral 복구, 새 음식 선택

### 2.5 콤보 시스템

- 정답 연속 터치 시 콤보 카운트 증가
- 콤보 보너스: 콤보수 x 2 추가 성장치
- 3콤보 이상 시 메시지 표시: "N 콤보! 아기가 아주 좋아해요! ✨"
- 2콤보 이상 시 화면에 "🌟 N 콤보!" 표시 (화면 중앙 52% 위치)
- 오답/독사과 터치 시 콤보 초기화

### 2.6 원하는 음식 시스템

- 아기가 원하는 음식은 말풍선으로 표시 (화면 우상단 부근)
- 변경 주기: 10초 (`wantChangeInterval`)
- 타이머 40% 경과 시 "기다려..." 표시
- 타이머 70% 경과 시 "빨리!" 경고 표시
- 타이머 만료 시 싫음 게이지 증가 후 새 음식 선택

## 3. 캐릭터

### 3.1 엄마 백설공주

- **에셋**: `assets/snow-white-mom.svg` (SVG 이미지)
- **폴백**: SVG 로딩 실패 시 👸 이모지 + "백설공주" 텍스트
- **렌더링 크기**: 100 x 150px (scale 적용)
- **위치**: 타이틀 씬 - 화면 상단 20%, 게임 씬 - 화면 상단 12%
- **애니메이션**: 상하 바운스 (sin파)

### 3.2 아기 (4종 감정 표현)

| 감정 | 에셋 파일 | 폴백 이모지 |
|------|----------|-----------|
| neutral | `baby-snow-white.svg` | 👶 |
| happy | `baby-snow-white-happy.svg` | 😊 |
| sad | `baby-snow-white-sad.svg` | 😢 |
| angry | `baby-snow-white-angry.svg` | 😠 |

- **크기**: 20 + (성장비율 x 35) px
- **위치**: 배 원(belly) 중앙 (화면 세로 35%)
- **효과**: 성장비율 50% 이상 시 오버레이 글로우, 60% 이상 시 🌸 스파클
- **흔들림**: 오답/독사과 시 좌우 흔들림 (sin 30Hz, 감쇠)

## 4. 씬 구성

### 4.1 TitleScene (타이틀 화면)

- **배경**: 어두운 숲 그라디언트 (#0a1a0a ~ #2a4a2a)
- **장식**: 나무 실루엣, 떠다니는 낙엽/나비/꽃/별 (25개)
- **구성 요소**:
  - 엄마 백설공주 (scale 1.3, 바운스 애니메이션)
  - 주변 회전 스파클 (✨🌟🌸🦋🍃)
  - 제목: "👑 백설공주 아기 키우기 👑"
  - 부제: "숲속 동물들이 음식을 가져다줘요!" / "좋아하는 음식을 골라주세요~"
  - 아기 미리보기 (초록 글로우 원 안, neutral, 성장비율 0.3)
  - 시작 버튼: "🌿 시작하기 🌿" (초록 그라디언트, 맥동 효과)
  - 하단 힌트: "동물을 터치해서 음식을 주세요!"

### 4.2 GameScene (게임 플레이)

- **배경**: 어두운 숲 + 나무 실루엣 + 떠다니는 낙엽(12개) + 반딧불(8개)
- **UI 구성**:
  - 상단: 성장 바 (초록 그라디언트, 퍼센트 표시)
  - 상단 아래: 싫음 바 (보라색 그라디언트, 70% 이상 시 깜빡임)
  - 엄마 백설공주 (화면 상단 12%)
  - 원하는 음식 말풍선 (엄마 우측)
  - 배 원 + 아기 (화면 중앙 35%)
  - 성장 단계 텍스트
  - 콤보 표시
  - 동물 영역 안내: "🐾 동물을 터치해서 음식을 주세요!" (62%)
  - 동물들 (화면 하단 68~86% 영역)
- **파티클 시스템**: 정답/오답/독사과 시 이펙트
- **메시지 시스템**: 이벤트 알림

### 4.3 BornScene (아기 탄생)

- **배경**: 석양 숲 그라디언트 (#1a1a2e ~ #FFB347)
- **축하 파티클**: 꽃/나비/별 25개 (하단에서 상단으로 부유)
- **페이즈 연출** (시간 기반):
  - 0.5초: 중앙 빛 효과
  - 1.5초: 엄마 + 아기(happy) 등장
  - 2.5초: "🎉 아기가 태어났어요! 🎉" + "숲속 동물 친구들이 축하해요 🌸"
  - 3.5초: 통계 표시 (N번째 아기, 총 먹인 횟수, 정확도)
  - 4.0초: "다시 하기 🌸" 버튼

- **데이터 저장**: `localStorage`에 누적 출생 횟수 저장 (키: `snow-white-baby-born-count`)
- **통계 항목**: 총 먹인 횟수 (`totalFed`), 정답 횟수 (`correctFed`), 정확도 (`accuracy`)

## 5. 설정값 (config.js)

### 게임 밸런스

| 설정 | 값 | 설명 |
|------|----|------|
| `maxGrowth` | 100 | 최대 성장치 (탄생 조건) |
| `growthPerCorrect` | 8 | 정답 시 기본 성장량 |
| `growthPerWrong` | -2 | 오답 시 성장 감소량 |
| `dislikePerWrong` | 20 | 오답 시 싫음 증가량 |
| `dislikeDecay` | 3 | 싫음 자연 감소 (초당) |
| `maxDislike` | 100 | 싫음 최대치 (저주 발동 기준) |
| `poisonPenalty` | -15 | 독사과 성장 페널티 |
| `poisonDislike` | 40 | 독사과 싫음 페널티 |
| `curseDuration` | 5 | 마법 거울 저주 지속 시간 (초) |
| `wantChangeInterval` | 10 | 원하는 음식 변경 주기 (초) |
| `comboBonus` | 2 | 콤보당 추가 성장량 |
| `maxAnimals` | 5 | 최대 동시 동물 수 |
| `minAnimals` | 3 | 최소 유지 동물 수 |
| `animalSpawnInterval` | 2 | 동물 스폰 간격 (초) |
| `crowInterval` | 15 | 마녀 출현 간격 (초) |
| `wantedFoodChance` | 0.4 | 원하는 음식 확률 (config 선언값, 실제 코드는 0.5) |

### 마녀 설정

| 설정 | 값 |
|------|----|
| 이동 속도 | 70 |
| 크기 | 55 |
| 체류 시간 | 5초 |

## 6. 기술 스택

| 항목 | 기술 |
|------|------|
| 렌더링 | HTML5 Canvas 2D (`<canvas>`) |
| 모듈 시스템 | ES Modules (`type="module"`) |
| 캐릭터 에셋 | SVG 이미지 (5종), 이모지 폴백 |
| 게임 루프 | `requestAnimationFrame` (dt 기반, 최대 50ms 캡) |
| 입력 처리 | `shared/Input.js` (터치/클릭 추상화) |
| 파티클 | `shared/ParticleSystem.js` |
| 메시지 UI | `shared/ui/Message.js` |
| 화면 전환 | Fade in/out (alpha 기반, 속도 2.5) |
| 데이터 저장 | `localStorage` (출생 횟수) |
| 반응형 | 전체 화면, `resize` 이벤트 대응, safe-area-inset 지원 |
| 모바일 지원 | PWA 메타 태그, touch-action: none, user-select: none |

## 7. 파일 구조

```
snow-white-baby/
├── index.html                      # 진입점 HTML (에러 표시, 모듈 로드)
├── assets/
│   ├── snow-white-mom.svg          # 엄마 백설공주 SVG
│   ├── baby-snow-white.svg         # 아기 (neutral)
│   ├── baby-snow-white-happy.svg   # 아기 (happy)
│   ├── baby-snow-white-sad.svg     # 아기 (sad)
│   └── baby-snow-white-angry.svg   # 아기 (angry)
└── js/
    ├── main.js                     # 메인 엔트리, 게임 루프, 씬 관리, 전환
    ├── config.js                   # 모든 게임 설정값 (음식, 동물, 성장, 밸런스)
    ├── draw-snow-white.js          # SVG 에셋 로딩 및 캐릭터 렌더링 함수
    └── scene/
        ├── TitleScene.js           # 타이틀 화면
        ├── GameScene.js            # 메인 게임 플레이
        └── BornScene.js            # 아기 탄생 결과 화면

shared/                             # 프로젝트 공통 모듈 (상위 디렉토리)
├── Input.js                        # 터치/클릭 입력 처리
├── ParticleSystem.js               # 파티클 이펙트
└── ui/
    └── Message.js                  # 화면 메시지 표시
```
