# 인어공주 아기 키우기 - 게임 스펙 문서

## 1. 게임 개요

- **이름**: 인어공주 아기 키우기
- **컨셉**: 바다 속 물고기들이 음식을 들고 헤엄쳐 지나가면, 아기가 원하는 음식을 가진 물고기를 낚아채서 아기를 성장시키는 터치 기반 캐주얼 게임
- **타겟 유저**: 어린이 (넉넉한 터치 판정, 이모지 기반 UI, 한글 안내)
- **테마**: 디즈니풍 인어공주 + 심해 분위기 (깊은 바다 배경, 거품, 수면 빛줄기)
- **핵심 차별점**: 백설공주 게임은 "멈춘 동물 고르기"이나, 인어공주 게임은 "헤엄치는 물고기 낚아채기" — 타이밍 + 선택의 이중 재미

---

## 2. 게임 메카닉

### 2.1 바다 낚시 시스템

바다 속 생물들이 화면을 가로질러 헤엄치며 음식을 들고 지나간다. 플레이어는 원하는 음식을 든 생물을 터치하여 낚아챈다.

**핵심 차이: 통과형 이동**
- 백설공주: 동물이 목표 지점에 멈추고 대기 → 플레이어가 골라서 탭
- 인어공주: 물고기가 **멈추지 않고 계속 헤엄** → 플레이어가 타이밍에 맞춰 탭

**바다 생물 종류 (4종)**

| 동물 | 이모지 | 이동속도 | 크기 | 유영 패턴 |
|------|--------|---------|------|----------|
| 열대어 | 🐠 | 120 | 40 | 빠르고 직선 유영 |
| 거북이 | 🐢 | 50 | 65 | 느리고 위아래 출렁 |
| 오징어 | 🦑 | 100 | 45 | 지그재그 분사 |
| 게 | 🦀 | 70 | 50 | 바닥 근처 옆걸음 |

**스폰 규칙**
- 화면 좌측(-50px) 또는 우측(w+50px)에서 출현
- y좌표: 화면 하단 60~90% 영역 (바다 영역)
- 스폰 간격: 2초 (`animalSpawnInterval`)
- 최대 동시 출현: 5마리 (`maxAnimals`)
- 최소 유지: 3마리 (`minAnimals`)
- 화면 반대편으로 나가면 자동 제거 (통과형)

**유영 방식**
- 기본: 스폰 방향 → 반대쪽으로 수평 이동
- 사인파 적용: y축으로 물결치듯 유영 (amplitude, frequency 동물별 상이)
- 물리 충돌: 물고기끼리 부딪히면 튕겨남 (기존 충돌 물리 재활용)
- 경계 반발: 상하 화면 경계에서 튕김

**음식 선택**
- 50% 확률로 아기가 원하는 음식을 가져옴
- 나머지는 12종 음식 중 랜덤

### 2.2 바다마녀 시스템 (먹물 유혹)

- **등장 주기**: 15초마다 (`crowInterval`)
- **외형**: 문어 이모지(🐙), 보라색 배경 원, 보라색 경고 테두리 맥동 효과
- **유혹 메카닉**: 바다마녀는 아기가 현재 원하는 음식의 이모지를 촉수에 들고 유혹
- **이동**: 다른 물고기와 같이 헤엄치며 지나감 (통과형)
- **이동 속도**: 60 / 크기: 55

**바다마녀 터치 시 페널티**
- 성장치 -15 (`poisonPenalty`)
- 싫음 게이지 +40 (`poisonDislike`)
- 콤보 초기화
- 아기 감정: angry (2초간)
- 먹물 파티클(검은색) + "먹물이다! 🖤" 메시지

### 2.3 성장 시스템

- **최대 성장치**: 100 (`maxGrowth`)
- 성장치가 100에 도달하면 아기 탄생 (Born 씬으로 전환)

**정답 음식 터치 시**
- 성장치 + (8 + 콤보수 x 2)
- 싫음 게이지 -10
- 아기 감정: happy (1.5초)
- 물방울 파티클(파란색) + 별 이펙트 + "+N 🫧" 텍스트
- 낚아채기 애니메이션: 물고기가 위로 쭉 끌려올라감
- 새로운 원하는 음식 선택

**오답 음식 터치 시**
- 성장치 -2
- 싫음 게이지 +20
- 콤보 초기화
- 아기 감정: sad (1.2초) + 흔들림 효과 (0.5초)
- 물고기가 "퉤!" 하고 빠르게 도망가는 모션

**성장 단계 (6단계)**

| 단계명 | 크기 배율 | 진행률 기준 | 설명 |
|--------|----------|-----------|------|
| 진주알 | 0.3 | 0% | 반짝이는 알이에요 |
| 올챙이 | 0.45 | 15% | 꼬물꼬물! |
| 치어 | 0.6 | 30% | 작은 지느러미가! |
| 꼬마인어 | 0.75 | 50% | 꼬리가 생겼어요! |
| 인어아기 | 0.9 | 70% | 노래를 흥얼거려요! |
| 공주님 | 1.0 | 90% | 바다의 새 공주! |

### 2.4 싫음 게이지 & 소용돌이 저주

- **최대 싫음**: 100 (`maxDislike`)
- **자연 감소**: 초당 3 (`dislikeDecay`)
- **원하는 음식 타이머 만료 시**: 싫음 +8, 새 음식 선택
- **싫음 100 도달 시**: 소용돌이 저주 발동

**소용돌이 저주**
- 지속 시간: 5초 (`curseDuration`)
- 효과: 입력 차단 (터치 무시), 물고기 스폰/이동 정지
- 시각 효과: 어두운 보라/남색 오버레이, 소용돌이 회전 이펙트, 저주 이모지(🌀🐙⚡🔱) 회전, 카운트다운 표시
- 종료 시: 싫음 0으로 초기화, 감정 neutral 복구, 새 음식 선택

### 2.5 콤보 시스템

- 정답 연속 터치 시 콤보 카운트 증가
- 콤보 보너스: 콤보수 x 2 추가 성장치
- 3콤보 이상 시 메시지 표시: "N 콤보! 아기가 아주 좋아해요! 🫧"
- 2콤보 이상 시 화면에 "🌊 N 콤보!" 표시
- 오답/먹물 터치 시 콤보 초기화

### 2.6 원하는 음식 시스템

- 아기가 원하는 음식은 말풍선으로 표시 (엄마 인어공주 옆)
- 변경 주기: 10초 (`wantChangeInterval`)
- 타이머 40% 경과 시 "기다려..." 표시
- 타이머 70% 경과 시 "빨리!" 경고 표시
- 타이머 만료 시 싫음 게이지 증가 후 새 음식 선택

## 3. 캐릭터

### 3.1 엄마 인어공주

- **디자인**: 빨간 머리 (긴 웨이브) + 초록 꼬리 + 보라 조개 탑
- **에셋**: `assets/mermaid-mom.svg` (SVG 이미지)
- **폴백**: SVG 로딩 실패 시 🧜‍♀️ 이모지 + "인어공주" 텍스트
- **렌더링 크기**: 100 x 150px (scale 적용)
- **위치**: 타이틀 씬 - 화면 상단 20%, 게임 씬 - 화면 상단 12%
- **애니메이션**: 상하 바운스 (물에 둥실 떠있는 느낌, sin파)

### 3.2 아기 (4종 감정 표현)

| 감정 | 에셋 파일 | 폴백 이모지 |
|------|----------|-----------|
| neutral | `baby-mermaid.svg` | 👶 |
| happy | `baby-mermaid-happy.svg` | 😊 |
| sad | `baby-mermaid-sad.svg` | 😢 |
| angry | `baby-mermaid-angry.svg` | 😠 |

- **디자인**: 둥근 얼굴 + 작은 꼬리 (성장에 따라 변화)
- **크기**: 20 + (성장비율 x 35) px
- **위치**: 배(산호 둥지) 중앙 (화면 세로 35%)
- **효과**: 성장비율 50% 이상 시 오버레이 글로우, 60% 이상 시 🫧 스파클
- **흔들림**: 오답/먹물 시 좌우 흔들림

## 4. 씬 구성

### 4.1 TitleScene (타이틀 화면)

- **배경**: 심해 그라디언트 (#0a0a2e ~ #1a3a5a)
- **장식**: 산호 실루엣, 떠다니는 거품/해파리/물고기/별 (25개)
- **구성 요소**:
  - 엄마 인어공주 (scale 1.3, 떠다니는 애니메이션)
  - 주변 회전 스파클 (🫧🌊🐚🦀✨)
  - 제목: "🔱 인어공주 아기 키우기 🔱"
  - 부제: "바다 속 물고기가 음식을 가져다줘요!" / "좋아하는 음식을 낚아채세요~"
  - 아기 미리보기 (청록 글로우 원 안, neutral, 성장비율 0.3)
  - 시작 버튼: "🌊 시작하기 🌊" (남색/청록 그라디언트, 맥동 효과)
  - 하단 힌트: "물고기를 터치해서 음식을 낚아채세요!"

### 4.2 GameScene (게임 플레이)

- **배경**: 심해 그라디언트 + 산호/해초 실루엣 + 상승 거품(12개) + 수면 빛줄기(코스틱, 8개)
- **수면 빛줄기**: 화면 상단에서 아래로 대각선 빛줄기 (느리게 흔들림, 낮은 투명도)
- **UI 구성**:
  - 상단: 성장 바 (청록 그라디언트, 퍼센트 표시)
  - 상단 아래: 싫음 바 (보라색 그라디언트, 70% 이상 시 깜빡임)
  - 엄마 인어공주 (화면 상단 12%)
  - 원하는 음식 말풍선 (엄마 우측)
  - 산호 둥지 + 아기 (화면 중앙 35%)
  - 성장 단계 텍스트
  - 콤보 표시
  - 낚시 안내: "🐠 물고기를 터치해서 음식을 낚아채세요!" (58%)
  - 물고기들 (화면 하단 60~90% 영역, 계속 헤엄침)
- **파티클 시스템**: 정답 시 물방울, 오답 시 거품, 먹물 시 검은 잉크
- **메시지 시스템**: 이벤트 알림

### 4.3 BornScene (아기 탄생)

- **배경**: 심해 → 밝은 바다 그라디언트 (#1a1a2e ~ #4FC3F7)
- **축하 파티클**: 거품/조개/별/진주 25개 (하단에서 상단으로 부유)
- **페이즈 연출**:
  - 0.5초: 중앙 빛 효과 (바다빛)
  - 1.5초: 엄마 + 아기(happy) 등장
  - 2.5초: "🎉 아기가 태어났어요! 🎉" + "바다 친구들이 축하해요 🫧"
  - 3.5초: 통계 표시
  - 4.0초: "다시 하기 🌊" 버튼

- **데이터 저장**: `localStorage`에 누적 출생 횟수 저장 (키: `mermaid-baby-born-count`)

## 6. 설정값 (config.js)

### 게임 밸런스

| 설정 | 값 | 설명 |
|------|----|------|
| `maxGrowth` | 100 | 최대 성장치 (탄생 조건) |
| `growthPerCorrect` | 8 | 정답 시 기본 성장량 |
| `growthPerWrong` | -2 | 오답 시 성장 감소량 |
| `dislikePerWrong` | 20 | 오답 시 싫음 증가량 |
| `dislikeDecay` | 3 | 싫음 자연 감소 (초당) |
| `maxDislike` | 100 | 싫음 최대치 (저주 발동 기준) |
| `poisonPenalty` | -15 | 먹물 성장 페널티 |
| `poisonDislike` | 40 | 먹물 싫음 페널티 |
| `curseDuration` | 5 | 소용돌이 저주 지속 시간 (초) |
| `wantChangeInterval` | 10 | 원하는 음식 변경 주기 (초) |
| `comboBonus` | 2 | 콤보당 추가 성장량 |
| `maxAnimals` | 5 | 최대 동시 물고기 수 |
| `minAnimals` | 3 | 최소 유지 물고기 수 |
| `animalSpawnInterval` | 2 | 물고기 스폰 간격 (초) |
| `crowInterval` | 15 | 바다마녀 출현 간격 (초) |
