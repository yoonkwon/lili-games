# 꼬꼬닭 알 낳기 - 게임 스펙 문서

## 개요
- **장르**: 캐주얼 디펜스 게임
- **타겟**: 5~6세 아동 (응애/동생), 초등학생 (언니), 성인 (엄마)
- **플랫폼**: 웹 (HTML5 Canvas), PWA 지원
- **기술**: ES Modules, Canvas 2D, Web Audio API, SpriteCache 프리렌더링
- **배포**: GitHub Pages (https://yoonkwon.github.io/lili-games/)

---

## 핵심 메카닉

### 1. 게이지 → 알 생산
| 항목 | 값 |
|------|-----|
| 게이지 칸 수 | 난이도별 2~3칸 |
| 채우는 방법 | 화면 터치/클릭 1회 = 1칸 |
| 게이지 만땅 시 | 알 1개 생성 → 둥지에서 대기 → 바구니로 이동 |
| 겨울 스테이지 | 게이지 자동 감소 (0.15/초) |
| 둥지 점거 | 천적이 둥지에 있으면 알 생산 불가 |

### 2. HP 시스템
| 항목 | 설명 |
|------|------|
| 최대 HP | 난이도별 10~15 |
| 알 도둑질 피해 | 도둑질 양에 비례: 1개→HP-1, 2~3개→HP-2, 4+→HP-3 |
| 직접 공격 피해 | HP -3 (알/병아리 없을 때) |
| 병아리 피납 피해 | HP -3 |
| HP = 0 | 게임오버 |

### 3. 콤보 시스템
- 0.6초 이내 연속 터치 시 콤보 누적
- 콤보 높을수록 황금알 확률 증가
- 최대 콤보 기록 → 점수 반영

### 4. 황금알
- 알 3개 가치 (일반 1개 vs 황금 3개)
- 확률: 난이도별 기본 5~20% + 콤보 보너스
- 황금알 확률 상한: 30%

## 천적 시스템

### 천적 종류
| 종류 | 속도 | 터치 횟수 | 도둑질 배율 | 이동패턴 |
|------|------|-----------|-------------|----------|
| 여우 🦊 | 100~130 | 1회 | ×0.5 | 직선 |
| 족제비 🐾 | 70~95 | 1회 | ×1.0 | 지그재그 |
| 너구리 🦝 | 40~60 | 3회 | ×1.5 | 직선 |

### 스폰 규칙
- **유예기간**: 게임 시작 후 3초간 천적 없음
- **스폰 간격**: `max(최소간격, 기본간격 - 알수 × 가속비율)` (알이 많을수록 빈번)
- **최소 스폰 간격**: 난이도별 1.5~2.5초
- **동시 출현**: 난이도별 최대 2~5마리
- **버스트 스폰**: 확률적으로 한번에 2마리 동시 출현 (10~40%)
- **행동**: 좌/우에서 등장 → 둥지로 접근 → 도둑질 → 도주

### 천적 행동 우선순위
1. 병아리 포획 (가장 가까운 병아리 잡아감)
2. 알 도둑질 (바구니에서 알 뺏기)
3. 직접 공격 (알/병아리 없을 때 엄마닭 공격)

### 난이도별 천적 강화
- **속도 배율**: 0.6x ~ 1.8x
- **도둑질 배율**: 0.5x ~ 2.5x
- **터치 배율**: 1x ~ 2x (너구리 기준 3→6회)
- **도둑질 지연**: 0.5초 (기본) → 0.3초 (엄마)

## 방어 시스템

### 강아지 소환
두 마리의 가족 강아지를 전략적으로 소환:

| 강아지 | 속도 | 순찰범위 | 퇴치력 | 특성 |
|--------|------|----------|--------|------|
| 보리 🐕 (검은 차우차우) | 80 | 120px | 3 (즉시 퇴치) | 느리지만 강력 |
| 좁쌀이 🐶 (점박이) | 180 | 250px | 2 (다회 공격) | 빠르고 넓은 범위 |

- **충전**: 알을 모을수록 각 강아지 게이지 충전 (난이도별 8~14알)
- **지속시간**: 난이도별 8~18초
- **3번째 버튼 - 힐**: 충전된 게이지 소모 → HP +2 회복

### 병아리 방어
- 병아리가 성장하면 (size ≥ 0.5) 자동으로 **여우만** 방어
- 3마리 이상: 족제비도 방어, 5마리 이상: 모든 천적 방어
- 방어 후 8초 쿨다운
- 🌸 꽃 모자 해금 시 쿨다운 2배 빠르게 감소

## 스테이지 시스템

알을 모을수록 배경이 변하며 고유 효과 발동 (TARGET_EGGS / 5 단위):

| 스테이지 | 테마 | 효과 |
|----------|------|------|
| 1 | 🌸 봄 농장 | 없음 (기본) |
| 2 | ☀️ 여름 들판 | 천적 속도 ×1.2 |
| 3 | 🍂 가을 숲 | 15초마다 바람 (최근 낳은 알 1개 날림) |
| 4 | ❄️ 겨울 눈밭 | 게이지 자동 감소 |
| 5 | 🌈 무지개 마을 | 20초마다 랜덤 보너스 (알+2 / HP+1 / 강아지충전+5) |

## 모자 해금 & 능력

알 모으기 진행률에 따라 엄마닭에게 모자가 장착되며 영구 능력 부여:

| 진행률 | 모자 | 능력 |
|--------|------|------|
| 40% | 👑 왕관 | 모든 알 가치 +1 |
| 65% | 🎀 리본 | 게이지 필요 탭 1칸 감소 |
| 85% | 🌸 꽃 | 병아리 방어 쿨다운 2배 빠름 |
